Media Fragments for HTML Media Elements

Using this polyfill, one can link to media elements on a web page using fragment identifiers and media fragments, using URLs like http://example.org/podcast.html#episode1&t=01:23.

Audio Example

Annotations

  • Incredible!
  • Amazing!
  • Awesome!

Code

<audio id=audio controls> <source src=rolling.oga type=audio/ogg> <source src=rolling.mp3 type=audio/mpeg> </audio>

The music was generated by interpreting the output of the following C program as unsigned 8-bit mono audio at a sample rate of 8000hz. More programs like this exist.

#include <stdio.h> main() { long int t; int sample; for(t=0;;t++) { sample = ( ((t*9 & t>>4) | (t*5 & t>>7) | (t*3 & t/0x400)) -1 ); fputc(sample, stdout); } }

Code

<video id=video src=ball.ogv controls></video>

The video was generated using GStreamer:

gst-launch-0.10 videotestsrc pattern=ball ! theoraenc ! oggmux ! filesink location=ball.ogv
